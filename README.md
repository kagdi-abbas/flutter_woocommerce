# Flutter Woocommerce App  
Flutter App to build mobile app for an exisiting Woocommerce store.  
This is a starter project inspired by Snippet Coder's Woocommerce tutorial.  
https://www.youtube.com/watch?v=6iQjW2nLgIk&list=PL7zgwanvi8_M6sq2qbGbT5XWhGoUYwa3T

Prerequisites  
  
- Flutter SDK (>= 3.x.x)
- Dart (included with Flutter)
- Android Studio / Xcode (for emulator or device deployment)
- Git

Verify Flutter setup:  
bashflutter doctor

## Getting Started
1. Clone the repository
  bashgit clone https://github.com/kagdi-abbas/flutter_woocommerce.git  
  cd repository-name  
2. Install dependencies  
  bashflutter pub get  
3. Run the app  
  bashflutter run
  
## Dependencies
 - flutter_svg
 - flutter_staggered_grid_view
 - http
 - snippet_coder_utils
 - provider
 - api_cache_manager
